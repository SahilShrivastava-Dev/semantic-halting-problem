# Welcome to the Semantic Entropy Convergence Wiki

This wiki documents the mathematical architecture and use cases for dynamically halting Large Language Model (LLM) agents using Semantic Entropy.

## Core Concepts
- **State Embedding**: Converting agent outputs into vectors.
- **Distance Calculation**: Tracking the delta (cosine distance) between iterations.
- **The Contraction Operator**: Watching the gradient of the distance.
- **The Convergence Threshold**: The epsilon point at which the graph terminates natively.

See the [Dubai Real Estate Example](Dubai-Real-Estate-Example.md) to see this applied in practice.
